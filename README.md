# 📂 File Sharing App

A simple and secure file sharing web application that allows users to upload files and share them using a generated link.

---

## 🚀 Features

- 📤 Upload files easily
- 🔗 Generate shareable download links
- 📥 Download files using link
- 🔐 Secure file access
- ⏳ (Optional) File expiration support
- 📱 Responsive UI

---

## 🛠️ Tech Stack

### Frontend
- React.js
- HTML, CSS, JavaScript
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB
